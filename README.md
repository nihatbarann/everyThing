# EveryThing

A secure, modern business management platform built with **PHP (MVC + SOLID)**, **React (Vite)**, and **MySQL**.

## Features

- **One-Step Installation Wizard** вЂ” Browser-based setup that configures the database and creates the default admin user.
- **JWT Authentication** вЂ” Stateless, secure token-based auth.
- **Role-Based Access Control (RBAC)** вЂ” Admin controls which menus/features each user can access.
- **Self-Optimizing Database** вЂ” Automated `OPTIMIZE TABLE` routines keep MySQL healthy as data grows.
- **Responsive UI** вЂ” Modern dark glassmorphic design that works on desktop and mobile.
- **No Public Registration** вЂ” All users are created by the admin.

## Prerequisites

- PHP 8.0+
- MySQL 8.0+ or MariaDB 10.5+
- Apache 2.4+ with `mod_rewrite` enabled
- Node.js 18+ and npm

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/your-username/everything.git
cd everything
```

### 2. Configure Apache (Virtual Host)

Point your Apache document root to the project folder and enable `mod_rewrite`.  
The `backend/.htaccess` handles API routing automatically.

### 3. Install frontend dependencies

```bash
cd frontend
npm install
npm run build
```

### 4. Set the JWT secret (Recommended for production)

Add this line to your Apache VirtualHost config or `backend/.htaccess`:

```apache
SetEnv JWT_SECRET "your-long-random-secret-here"
```

### 5. Run the installation wizard

Open your browser and navigate to your domain or `localhost`. The installation wizard will appear automatically and guide you through:

1. Entering database credentials
2. Testing the connection
3. Creating the database and tables
4. Setting up the default admin user

Default login after installation:
- **Username:** `everything`
- **Password:** `admin`

## Security Notes

- `backend/config/database.php` is listed in `.gitignore` and is **never committed**.
- The JWT secret is read from the `JWT_SECRET` environment variable.
- All SQL queries use **prepared statements** (PDO).
- Passwords are hashed with **bcrypt** (`password_hash` / `password_verify`).
